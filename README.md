# Frontend Practice Projects

A collection of beginner frontend projects built while learning HTML, CSS, and JavaScript.

## 🎵 Spotify Clone
A Spotify-inspired music player interface.

**Features**
- Playlist UI
- Audio controls
- Responsive design

## ✂️ Rock Paper Scissors
Classic game built using JavaScript.

**Features**
- Random computer choice
- Score tracking
- Interactive gameplay

## ❌ Tic Tac Toe
Two-player game with win detection.

**Features**
- Winner detection
- Draw handling
- Reset functionality

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript

---

These projects were built while learning frontend web development fundamentals.
